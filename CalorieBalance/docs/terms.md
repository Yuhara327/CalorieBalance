# 利用規約

この利用規約（以下、「本規約」といいます）は、Soichiro Yuhara（以下、「当方」といいます）が提供するアプリ「Calorie Balance」（以下、「本アプリ」といいます）の利用条件を定めるものです。

## 1. サービスの概要
本アプリは、日々のカロリー収支に基づき未来の体重を予測・管理するツールです。本アプリが提供する予測データは、学術的な計算式に基づくシミュレーションであり、医療的な診断や助言を代替するものではありません。

## 2. 免責事項
- ユーザーが本アプリの情報を用いて行う一切の行為について、当方は何ら責任を負うものではありません。
- ダイエットや健康管理は個人の責任において行い、必要に応じて専門医の診断を受けてください。
- 本アプリの利用により発生した損害について、当方の過失（重過失を除く）による場合は、直近1ヶ月の利用料金を賠償の上限とします。

## 3. サブスクリプション（Proプラン）
- 本アプリの一部機能は、有料のサブスクリプション（以下、「Proプラン」）への加入が必要です。
- 決済はApple Inc.の提供するアプリ内課金システムを通じて行われます。
- 解約はユーザー自身がApple IDの設定画面より行うものとし、当方で解約手続きを代行することはできません。

## 4. 禁止事項
ユーザーは、本アプリの利用にあたり、以下の行為をしてはなりません。
- 本アプリの逆コンパイル、リバースエンジニアリング、またはこれに類する行為。
- 運営を妨害する恐れのある不正アクセス。

## 5. 規約の変更
当方は、必要と判断した場合には、ユーザーに通知することなくいつでも本規約を変更することができるものとします。
