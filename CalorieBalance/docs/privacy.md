# プライバシーポリシー

Soichiro Yuhara（以下、「当方」といいます）は、スマートフォン用アプリケーション「Calorie Balance」（以下、「本アプリ」といいます）における、ユーザーの個人情報の取扱いについて、以下のとおりプライバシーポリシー（以下、「本ポリシー」といいます）を定めます。

## 1. 収集する情報
本アプリでは、以下の情報を取得する場合があります。
- **身体情報**: 体重、摂取カロリー、睡眠時間、性別、年齢など。
- **ヘルスケアデータ**: ユーザーの同意を得た場合に限り、Apple HealthKitから取得する健康関連データ。
- **利用状況**: アプリの利用履歴やクラッシュレポート。

## 2. 利用目的
取得した情報は、以下の目的のためにのみ利用します。
- 体重予測アルゴリズムの計算およびグラフ表示
- 目標達成状況の可視化
- サービスの改善および不具合の修正

## 3. ヘルスケアデータの取り扱い
Apple HealthKitから取得したデータは、本アプリの機能提供（健康管理および分析）のためにのみ使用します。これらのデータを広告配信やマーケティング、第三者への販売に利用することはありません。

## 4. データの保存と管理
ユーザーが入力したデータ、およびHealthKitから取得したデータは、原則としてユーザーの端末内またはユーザー個人のiCloud領域に保存されます。当方がサーバー上で個人の身体情報を収集・蓄積することはありません。
